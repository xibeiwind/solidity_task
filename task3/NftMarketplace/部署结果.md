## 合约功能说明

### 1. MyToken (ERC20代币)
- **地址**: `0xB4Cb7334145Ff19d4E65d840989A7E88a4B4A103`
- **功能**: 示例ERC20代币，用于拍卖支付
- **用途**: 支持ERC20代币支付的拍卖场景

### 2. MyNFT (ERC721 NFT)
- **地址**: `0x22CC36A207EDFd32A25526E07aAdcFE6c5359f6f`
- **功能**: 示例ERC721 NFT合约
- **用途**: 拍卖的NFT资产

### 3. ChainlinkPriceOracle (价格预言机)
- **地址**: `0x9CA30fB9C74F2BCda7a44112CAE307686B99b945`
- **功能**: 集成Chainlink的价格预言机
- **用途**: 获取ETH和ERC20代币的实时价格

### 4. NFTAuctionFactory (拍卖工厂)
- **地址**: `0x4312E31AF8D312b1091a9d399469AF16969d1EB2`
- **功能**: 拍卖工厂合约，采用UUPS可升级代理模式
- **用途**: 创建和管理拍卖合约，支持协议费用管理

## 使用说明

### 1. 创建拍卖
通过工厂合约创建新的拍卖：
```javascript
const factory = await ethers.getContractAt("NFTAuctionFactory", "0x4312E31AF8D312b1091a9d399469AF16969d1EB2");
const tx = await factory.createAuction(sellerAddress);
```

### 2. 配置拍卖
设置价格预言机并开始拍卖：
```javascript
const auction = await ethers.getContractAt("SingleNFTAuction", auctionAddress);
await auction.setPriceOracle("0x9CA30fB9C74F2BCda7a44112CAE307686B99b945");
```

### 3. 参与拍卖
支持ETH和ERC20代币出价：
- **ETH出价**: 直接发送ETH到拍卖合约
- **ERC20出价**: 使用MyToken代币 (`0xB4Cb7334145Ff19d4E65d840989A7E88a4B4A103`)
